# How to install
<br>
- Unity->Window->Package Manager<br>
- Click "+" left corner<br>
- Add package from git URL<br>
- Insert <code>https://github.com/Kamison/com.unity.ide.windsurf.git</code><br>
- Add<br>
- Done
